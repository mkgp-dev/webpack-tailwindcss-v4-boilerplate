# webpack-tailwindcss-v4-boilerplate
A simple starter setup using Webpack and TailwindCSS v4 for fast and efficient front-end development.

## Features
- Configured [Webpack](https://webpack.js.org/) for bundling JavaScript, CSS, and assets
- Integrated [TailwindCSS v4](https://tailwindcss.com/) for utility-first styling
- Supports live development with hot reloading
- Includes basic project structure

## Usage
Fork or clone this repository and install the required libraries
```bash
npm install
```
Running in development
```bash
npm run dev
```
Building the project with **Webpack**
```bash
npm run build
```
